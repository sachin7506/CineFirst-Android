## 🚀 Features  
- Paginated user list from ReqRes.in API  
- Trending movies via TMDB API  
- Offline data sync with Room + WorkManager  

## 🛠 Tech Stack  
| Component       | Technology         |
|-----------------|--------------------|
| Architecture    | MVVM               |
| Network         | Retrofit           |
| Async           | Kotlin Flow        |
| Database        | Room               |
